Privacy Policy

Last Updated: November 17, 2025

Summary: Video Agent is a self-hosted tool. We do not collect, store, or share your personal data. All data processing happens on your own infrastructure using your own API keys.

1. Introduction

This Privacy Policy explains how Video Agent ("we," "our," or "the Service") handles information when you use our automated video creation tool.

Important: Video Agent is designed to be self-hosted. We do not operate centralized servers that collect your data.

2. Information We Do NOT Collect

Because Video Agent is self-hosted, we do not collect:

  • Personal identification information
  • Email addresses or contact information
  • Payment information
  • Usage analytics or tracking data
  • Content you create
  • API keys or credentials
  • Social media account information

3. Data Processing

When you use Video Agent, data is processed as follows:

Service Data Processed Purpose
Reddit API Public post content Source material for videos
AWS Bedrock Text prompts, scripts AI content generation
Revid.ai Video scripts Video rendering
Social Media APIs Videos, captions, metadata Content distribution

Note: All data processing uses YOUR API keys and accounts. Data is sent directly from your system to these third-party services.

4. Third-Party Services

Video Agent integrates with third-party services. Each service has its own privacy policy:

  • Reddit: Reddit Privacy Policy
  • AWS: AWS Privacy Notice
  • Revid.ai: Revid.ai Privacy Policy
  • TikTok: TikTok Privacy Policy
  • YouTube: Google Privacy Policy
  • Instagram: Instagram Privacy Policy

5. Data Storage

All data is stored locally on your system:

  • Configuration: Stored in .env file on your machine
  • Videos: Saved to output/videos/ directory
  • Metadata: Saved as JSON files locally
  • Logs: Stored locally in output/cost_log.json
  • Credentials: Stored in your .env file (never transmitted to us)

6. Data Security

You are responsible for securing your data:

  • Keep your .env file secure and never commit it to version control
  • Use strong passwords for all API accounts
  • Regularly rotate API keys and credentials
  • Keep your system and dependencies updated
  • Use secure network connections

7. API Keys and Credentials

Video Agent requires you to provide API keys for:

  • Reddit API
  • AWS Bedrock
  • Revid.ai
  • Social media platforms (optional)

Important: These credentials are stored locally on your machine and are never sent to us. You are responsible for keeping them secure.

8. Content You Create

Content created using Video Agent:

  • Is owned by you
  • Is stored on your local system
  • May be sent to third-party services (Revid.ai, social media platforms) based on your configuration
  • Is subject to the terms of service of platforms where you post it

9. Cookies and Tracking

Video Agent does not use cookies or tracking technologies. If you host these policy pages on a website, standard web server logs may be collected by your hosting provider.

10. Children's Privacy

Video Agent is not intended for use by children under 13. We do not knowingly collect information from children.

11. International Data Transfers

When you use third-party services through Video Agent, your data may be transferred to and processed in different countries. Each service's privacy policy governs these transfers.

12. Your Rights

Because all data is stored locally on your system, you have complete control:

  • Access: All data is in your output/ directory
  • Deletion: Delete files from your system at any time
  • Portability: All data is in standard formats (JSON, MP4)
  • Correction: Edit configuration files directly

13. Data Retention

You control data retention:

  • Videos and metadata are kept until you delete them
  • Logs are kept until you clear them
  • No automatic deletion or retention policies

14. Changes to Privacy Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated "Last Updated" date.

15. Open Source

Video Agent is open source software. You can review the code to verify our privacy practices:

  • GitHub: https://github.com/bilnels09/video-agent-policies
  • All code is publicly auditable
  • No hidden data collection

16. Compliance

As a self-hosted tool, compliance with data protection regulations (GDPR, CCPA, etc.) is your responsibility. We provide the tools; you control the data.

17. Third-Party Analytics

Video Agent does not include any analytics or telemetry. We do not track usage, errors, or any other metrics.
